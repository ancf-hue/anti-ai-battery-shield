# 🔋 Anti-AI Battery Shield
## 🔬 Почему 10–90%?
Литий-ионные аккумуляторы деградируют быстрее при высоком напряжении (>90%) и глубоком разряде (<10%). Поддержание в диапазоне 10–90% снижает химический износ катода и анода в **3–4 раза**, продлевая жизнь батареи на сотни циклов перезарядки.

---

## ✨ Ключевые механики

| Фича | Как работает | Польза |
|------|--------------|--------|
| 🔊 **Звуковой СБП-Оповеститель** | Web Audio API генерирует пульсирующие частоты при выходе за границы | Заставляет отреагировать даже со спящим телефоном |
| 🎨 **Неоновый Стресс-Холст** | Canvas-рендер с частицами и глитч-эффектами | Визуальный контроль из любой точки комнаты |
| 📊 **Офлайн-Калькулятор износа** | LocalStorage хранит историю, считает баллы деградации | Реальная статистика без отправки телеметрии в Google/Apple |
| 🧱 **Webhook для Умного Дома** | POST-запрос при 90% для отключения розетки (Tuya, Sonoff, Яндекс) | Автоматическая физическая защита батареи |

###  Принципы проекта
- ✅ **100% офлайн** — всё работает в браузере, 0 сетевых запросов (кроме опционального Webhook)
- ✅ **Нулевое знание** — история зарядки и настройки хранятся только в `localStorage`
- ✅ **Прозрачный код** — один `index.html`, никаких зависимостей, трекеров и аналитики
- ✅ **Бесплатно навсегда** — социальный проект для продления жизни устройств
